Detailed Description

The connectivity kernel is basically an aggregate of ItemVectors for vertices, faces, and halfedges.

This class contains no geometry information - only information about connectivitiy. Arguably it abstracts away the implementation from the ConnectivityKernel class making it possible, for instance, to use a different kernel.
