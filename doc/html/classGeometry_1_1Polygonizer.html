Detailed Description

Polygonizer is the class used to perform polygonization.

inline

Constructor of Polygonizer.

The first argument is the ImplicitFunction that we wish to polygonize. The second argument is the size of the polygonizing cell. The third arg. is the limit to how far away we will look for components of the implicit surface. the fourth argument indicates whether the polygonizing cell is a tetrahedron (true) or cube (false). The final argument indicates whether normals should be computed.
