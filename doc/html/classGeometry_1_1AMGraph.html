Detailed Description

AMGraph means adjacency map graph.

It is a simple graph class that is similar to an adjacency list. the difference is that the adjacency is given by a map from node id to edge id instead of a list. The reason for this is that we can still iterate over all adjacent nodes (the keys) but we can also find ids for the concrete edges. This class is not abstract but also not intended for direct usage. There are no attributes for nodes or edges and it is not possible to remove nodes or edges from instances of this class. Look to the derived AMGraph3D.
