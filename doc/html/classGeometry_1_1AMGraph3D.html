Detailed Description

AMGraph3D extends the AMGraph class by providing attributes for position, edge and node colors.

This is mostly for convenience as these attributes could be stored outside the class, but it gives us a concrete 3D graph structure that has many applications.

Disconnect nodes.

This operation removes the edge from the edge maps of the two formerly connected vertices, but the number of edges reported by the super class AMGraph is not decremented, so the edge is only invalidated. Call cleanup to finalize removal.
