Detailed Description

template<class T>
class Geometry::RGrid< T >

Regular voxel grid. 

    This class template can be instantiated and used directly.
    This is just a regular voxel grid stored as a linear array
    with functions to access its contents.

Construct a regular voxel grid.

Read/write access to voxel grid.

This is a non-const overloaded operator[]. In a regular grid, we have reserved room for all voxels in advance. Hence, it is possible to create a non-const operator[]. See AncestorGrid::operator[].
